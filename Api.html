¿Qué es esto?

La API de Rick y Morty es una API REST(ish) y GraphQL basada en el programa de televisión Rick y Morty. Tendrá acceso a cientos de personajes, imágenes, ubicaciones y episodios. La API de Rick y Morty está llena de información canónica como se ve en el programa de TV.
